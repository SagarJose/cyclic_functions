# cyclic_functions
To show mapping operation in cyclic functions in mathematical structures.

Sample input:

Enter Your List please :12

OUTPUT

A 1 = ('1', '2') 
      ('1', '2') 

A 2 = ('1', '2') 
      ('2', '1') 

        A 1     A 2     
        ___     ___
